About

This webpage is to demonstrate the various skill set that have been learned and developed throught the course. The four main topic that have been learned are; Git, Docker, Python/Flask, and finally Continuous Integration and Deployment.

Git

Git is a DevOps tool used for source code management. It is a free and open-source version control system used to handle small to very large projects efficiently. Git is also especially useful when multiple users are working on the same project by allowing the users a convenient area to upload files to.

Docker

Docker is an open platform for developing, shipping, and running applications. Docker allows you to lower the amount of time delay between writing the code and running it.

Python and Flask

Python and Flask are both essential coding languages that every coder should know to at least some degree. They allow a webpage developer to be able to get creative with the webpage since they gain different options.

Continuous Integration and Deployment

Continuous Integration and Deployment are two different methodologies when creating a software or webpage. Continuous Intergration consists when multiple people are updating the software frequesntly. Continuous Deployment consist of having the software be deployable at all times and be able to run whenever it is needed.
